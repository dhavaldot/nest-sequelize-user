# nestJS, sequelize, postgres, GraphQL - Combined Project
  
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Features

1. CRUD Operation with PostgresDB using Sequlize
2. Can use raw query(writing queries) to perform dynamic oprations.
3. Implemented GraphQL.
